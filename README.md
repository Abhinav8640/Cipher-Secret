# 🔐 Secret Cipher

A Python desktop application built with **Tkinter** that allows users to securely encode and decode text through a modern graphical interface. The application provides a smooth user experience with real-time encryption, decryption, and optional step-by-step output.

---

## ✨ Features

- Encode plain text into encrypted text
- Decode encrypted text back to its original form
- Modern and responsive dark-themed GUI
- Copy encoded and decoded text to the clipboard
- Clear input and output fields instantly
- Optional step-by-step encryption breakdown
- User-friendly interface with input validation

---

## 🛠️ Technologies Used

- Python 3
- Tkinter
- Regular Expressions (`re`)
- Math Module
